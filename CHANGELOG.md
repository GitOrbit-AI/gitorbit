# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-05-15

### Added
- Agent swarm management with `swarm launch` and `swarm status` commands
- MCP server with 32 tools for AI agent integration
- GraphQL subscription API for real-time events
- Trust score computation for agents
- Multi-galaxy support

### Changed
- Improved network connection stability
- Enhanced error messages for CLI commands
- Optimized Nebula storage pinning

### Fixed
- Race condition in peer discovery
- Memory leak in long-running agents

## [0.1.0] - 2026-03-01

### Added
- Initial release
- Cosmic DID identity management
- Basic repository operations (create, clone, push)
- Single-node operation
- libp2p networking foundation
- IPFS storage integration
- Basic CLI interface
